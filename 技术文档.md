# 抖音无人直播系统 — 技术文档

> 版本：V1.0
> 日期：2026年7月10日
> 用途：AI 生成口播话术 → TTS 合成语音 → 自动轮播 → 无人直播

---

## 一、项目概述

生产者-消费者架构的无人直播系统。后台线程持续预生成话术+音频，主线程无缝播放。支持多音色轮换、话题加权选择、AI 降级话术、BGM 循环、随机停顿模拟真人节奏。

> ⚠️ 本项目的练习意义大于实际意义。无人直播缺少真人互动，无法回答观众提问。

---

## 二、技术栈

| 层面 | 技术 | 说明 |
|------|------|------|
| AI 话术 | DeepSeek API（OpenAI SDK） | 自然口播话术生成 |
| TTS 配音 | edge-tts | 4 音色轮换 |
| 音频播放 | pygame | 人声 + BGM 双通道 |
| 并发 | threading + Queue | 生产者-消费者模式 |

---

## 三、系统架构

```
后台线程（producer）                主线程（consumer）
  ┌──────────────────┐              ┌──────────────────┐
  │ script_gen.py    │              │ player.py        │
  │ → AI 生成话术     │──→ Queue ──→│ → pygame 播放     │
  │ tts.py           │   (缓冲区)   │ → 随机停顿       │
  │ → 合成语音        │              │ → 取下一条        │
  └──────────────────┘              └──────────────────┘
       不停预生成                          无缝播放
```

### 核心模块

| 模块 | 职责 |
|------|------|
| script_gen.py | 知识库 + 话题加权 + DeepSeek 话术生成 + 降级话术 |
| tts.py | edge-tts 文字转语音 |
| player.py | pygame 播放 + BGM 循环 |
| main.py | 配置 + 生产者线程 + 主循环 |

---

## 四、话题管理

7 个话题，权重偏向转化类：

```
套餐（2.0） > 购买方式（2.0） > 位置（1.5） > 安全（1.2）
> 店铺/儿童成长/营业时间（1.0）
```

近 3 轮不重复，AI 失败自动降级为知识库原文拼接。

---

## 五、使用方式

```bash
python main.py        # 启动直播，Ctrl+C 停止
```
